# Сорока Ворона Landing Page

Статичний лендинг для продажу дитячої книги для пальчикового малювання.

Відкрийте `index.html` у браузері або опублікуйте всю папку на GitHub Pages/Netlify/Vercel.

Форма зараз показує локальне повідомлення про успіх. Для реальних заявок підключіть CRM, Telegram bot, Formspree, Make/Zapier або власний endpoint у `app.js`.
